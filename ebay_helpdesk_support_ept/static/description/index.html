eBay Helpdesk

OVERVIEW

After sales service is very important part of any business, it would be always necessary to respond customer queries / complaints with top priorities. In eBay Marketplace you need to check all the customer queries by login to your seller account which is really headache to check all messages manually time to time and respond. eBay Helpdesk app create helpdesk ticket in Odoo directly from eBay customer messages.

Request a Free Demo

Features

  • Creates Helpdesk Ticket by fetch email from email address set on eBay seller account.
  • Detect Order Number automatically from the email subject & set in Helpdesk ticket.
  • Allows to configure / set support team at eBay Instance or Marketplace.
