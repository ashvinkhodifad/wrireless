This module requires Username, Password and Stamps Integration Id
You can avail it from Stamps.com

After Installing this module create delivery method with provider Stamps and add your stamps.com configurations

Set price and create sale order line for shipment charges

After selecting delivery method click on the "set price" button to add new sale order line with charges

Create shipment with stamps for delivery order

After successfull validation of delivery order tracking refernce number and label returns

The tracking reference number and shipping label is attached in message box

Click on Tracking button to track the shipment. It will redirect to stamps.com

There are some requirements when sending Military Mail packages. For Military Mail Shipments the addition field is given in contact form "Is APO/FPO/DPO?".

Select this field If the shipment is to Military Base or Diplomatic Locations. Also in city field you have to add APO/FPO/DPO and select the state from Armed Forces Americas(AA)/Armed Forces Europe(AE)/Armed Forces Pacific(AP).
